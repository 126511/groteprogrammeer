Les 2 - Het begin

Het doel van deze les in het kunnen vragen aan de gebruiker om een waarde, die waar wij vervolgens iets mee kunnen doen.

Paragraaf 1: Het opzetten van programma

Maar voor we kunnen beginnen met het schrijven van opdrachten voor de computer moeten we eerst ons bestand opzetten. Er zijn twee dingen die je moet doen voordat je in C kan beginnen.

  • Het importeren van verschililende bibliotheken

    • C zichzelf is niet zo'n uitgebreide programeertaal, daarom moeten we voor de simpelste opdracht, het uitprinten van een stukje tekst, al een bibliotheek (library) importeren. Gelukkig is dit heel erg simpel:

      #include <stdio.h>

      Met #include zeg je tegen de computer dat het volgende wilt importeren. stdio.h staat voor StandardIO, het is dus een standaard library voor INPUT/OUTPUT. Het staat tussen haakjes zodat C weet dat het een verwijzing is naar een bestand dat stdio.h heet. Het bestandstype .h staat voor header, ofwel een library.

  • Het openen van een hoofdfunctie

    • Een functie is een verzameling van opdrachten. Denk bijvoorbeeld aan wandelen. Dat is een functie die bestaat uit opdrachten zoals: rechterbeen omhoog, vooruit en naar beneden en linkerbeen omhoog, vooruit en naar beneden. Elk programma moet een hoofdfunctie hebben. die noemen we main. Een functie kan een argument hebben: dat is een input-waarde, zoals bijvoorbeeld hoeveel stappen je loopt. Ook heeft een functie een return-waarde. Dat is vaak een geheel getal zoals 0 of 1. Als het 0 is, is de functie succesvol uitgevoerd. Als het 1 is, is er een error gekomen en is de functie vroegtijdig gestopt.

      int main(void)

      • int is de return-waarde; het staat voor integer (geheel getal). Void betekent dat main geen arugments nodig heeft. Dit is het standaard format voor een main-functie.
      • Daarnaast hebben we ook nog accolades { } nodig om de code die in de functie hoort te scheiden van die daarbuiten. We hebben nu dit bestand.

Paragraaf 2: Het uitprinten van tekst

Nu we weten hoe we een programma kunnen starten kunnen we ook ons programma runnen, ofwel uitvoeren. Druk maar op de groene pijl middel bovenaan het venster hierboven. In het donkerblauwe veld; de terminal, zie je dat het programma twee commands uitvoert:

  • clang (met wat extra tekst erachter): clang is een programma dat van de tekst in het bestand een vertaling maakt naar een taal die de computer begrijpt. Je hoeft niet te begrijpen waarom dat command werkt en wat al die extra arguments doen.
  • ./main: dit betekent, voer de vertaling uit. Clang heeft de vertaling main genoemd. Als de vertaling programma had geheten, was het command dus ./programma geweest. Deze main heeft niks te maken met het feit van onze hoofd-funtie main heet.

Als je het programma uitvoert zie je dat er niks gebeurt. Dat komt omdat onze functie (tussen de accolades) leeg is! Laten we dat snel is opvullen met een opdracht voor de computer. De eerste opdracht is printf. Deze opdracht neemt 1 argument: de tekst die je wilt printen, deze tekst moet je tussen aanhalingstekens zetten, dat kunnen dubbele (") of enkele (') aanhalingstekens zijn. Als je maar begint en eindigt met dezelfde. Als je goedemorgen wilt uitprinten krijg je dus printf("goedemorgen"). Aan het einde van een functie heb je nog 1 ding nodig: een puntkomma (;). Dit geeft voor het programma aan dat je functie nu klaar is. Dan krijg je dit: printf("goedemorgen");.

Aan het einde van een print-functie, zit niet standaard een nieuwe regel. Om een nieuwe regel (ENTER) te printen aan het einde van hetgeen dat je print gebruik je het teken \n. Een backslash (\) geeft aan de computer aan dat het teken erachter niet zomaar moet worden uitgeprint, maar dat het moet worden geformatteerd, dat is het anders uitprinten van wat het is. De manier waarop je goedemorgen uitprint is dus printf("goedemorgen\n");.

Paragraaf 3: Het vragen om input van een gebruiker

Vaak willen we dat ons programma niet zomaar iets uitprint, maar dat we een vragen om input aan de gebruiker en daar dan later iets mee doen. Voor het vragen van een input hebben we een nieuwe library nodig. Naast stdio.h hebben we nu cs50.h nodig. Hierin zit de opdracht get_string. Deze functie neemt als argument de tekst die je wilt uitprinten en dan de gebruiken in een terminal vraagt. Voer bijvoorbeeld het programma hieronder uit. Het vraagt je om twee gehele getallen, die het programma optelt en die waarde weer uitprint.

In het bestand hierboven wordt op regel 5, 6 en 8 een variabele. Met get_int (een functie om de gebruiker een integer te vragen) worden x en y gedefiniÃ«erd. Een variabele is een naam die je aan een waarde geeft. Een variabele begin je met het type waarde die je wilt opslaan, voorbeelden zijn int (geheel getal), string (stukje tekst), char (karakter) of een bool (waar/niet waar). In dit geval kiezen we voor een int, aangezien je twee getallen wilt optellen. Vervolgens maken we een nieuwe variabele: z, wat de som is van x en y. Je mag variabelen dus optellen alsof het echte getallen zijn.

Om een variabele uit te printen gebruik je % met een letter daarachter. Voor integers is dat i, voor strings is dat s. Binnen de haakjes achter printf, maar buiten de aanhalingsteken voeg je de naam van de variabele toe na een komma. Om x uit te printen doe je dus printf("%i\n", x);.
